**1. Create `src/components/editor/text-replace-panel.tsx`** (per D-01, D-04, D-14, D-15, D-16, D-17, D-18, D-19):

"use client" directive. Props: `{ fabricRef: RefObject<FabricCanvas | null> }`

Imports: useTextReplace hook, useEditorStore, Button, Slider, Select (from shadcn), Input, Label, TextRegion type.

Local state:
- `replacementText: string` (controlled input for new text)
- `targetLang: string` (selected translation language, default "")

The panel has 3 sections depending on state:

**Section A -- No regions detected yet (textRegions.length === 0 and no active IText):**
- Description text: "Detect text in the image, then select regions to replace or translate."
- "Detect Text" button (primary) -> calls handleDetectText
- Disabled when isProcessing, shows "Detecting..." during processing

**Section B -- Regions detected, user browsing (textRegions.length > 0):**
- Scrollable list of detected regions, each showing:
  - Truncated original text (max 40 chars with ellipsis)
  - Confidence badge (green >= 0.7, yellow >= 0.5, red < 0.5)
  - Bounding box dimensions in px (small text)
  - Click to select (calls setSelectedRegionIndex)
  - Selected region has blue left border highlight
- When a region is selected (selectedRegionIndex !== null):
  - **Replace section:**
    - Input field for replacement text (value: replacementText, onChange updates local state)
    - "Replace" button -> calls handleReplaceText(selectedRegionIndex, replacementText)
  - **Translate section** (per D-18, D-19):
    - Language dropdown with options: Korean (ko), English (en), Japanese (ja), Chinese (zh), Spanish (es), French (fr), German (de)
    - "Translate & Replace" button -> calls handleTranslateAndReplace(selectedRegionIndex, targetLang)
    - Disabled if no targetLang selected
  - Both buttons disabled when isProcessing
- "Re-detect" button (outline, small) to re-run OCR
- "Cancel" button -> handleCancelReplace

**Section C -- IText placed on canvas (textRegions.length === 0 and canvas has TEXT_REPLACE_TAG object):**
Refinement controls (per D-15, D-16, D-17):
- **Font Family:** dropdown with options -- Arial (sans-serif), Georgia (serif), Courier New (monospace), Verdana, Helvetica, Times New Roman
  - On change: get active object from canvas, if IText, set fontFamily, canvas.renderAll()
- **Font Size:** number input with +/- stepper buttons
  - On change: set fontSize on active IText
- **Color:** native color input (type="color")
  - On change: set fill on active IText
- **Position note:** "Drag the text to reposition. Use handles to resize."
- **"Apply" button** (primary) -> calls handleApplyText (per D-16, flattens text onto image)
- **"Cancel" button** (outline) -> handleCancelReplace

Detect which section to show:
- Check canvas for objects with TEXT_REPLACE_TAG to determine if we are in refinement mode
- Use a `hasReplacedText` computed from checking fabricRef.current?.getObjects()
