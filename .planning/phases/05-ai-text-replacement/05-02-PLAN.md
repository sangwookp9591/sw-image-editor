**2. Create `src/components/editor/hooks/use-text-replace.ts`:**

"use client" directive at top. Hook signature: `useTextReplace(fabricRef: RefObject<FabricCanvas | null>)`

Imports:
- `useCallback` from "react"
- `type RefObject` from "react"
- `type Canvas as FabricCanvas` from "fabric"
- `toast` from "sonner"
- `useEditorStore` from "./use-editor-store"
- `detectText, removeObject, translateText` from "@/app/actions/ai-image"
- `extractTextStyle, createMaskFromBbox` from "@/lib/ai/text-style"
- `type TextRegion` from "@/lib/ai/ocr"

**`handleDetectText` callback** (per D-01, D-02, D-03):
- Get canvas from fabricRef, guard null
- Get `{ isProcessing, setIsProcessing, setTextRegions, setSelectedRegionIndex }` from `useEditorStore.getState()`
- If isProcessing, return early
- setIsProcessing(true)
- Reset viewport transform to [1,0,0,1,0,0], export canvas as base64 PNG, restore viewport
- Call `const { regions } = await detectText(base64Image)`
- If regions.length === 0, toast.info("No text detected in this image") and return
- setTextRegions(regions)
- setSelectedRegionIndex(null)
- toast.success(`Detected ${regions.length} text region(s)`)
- In finally: setIsProcessing(false)
- Wrap in try/catch, toast.error on failure

**`handleReplaceText` callback** (per D-06, D-07, D-08, D-09, D-14):
Takes `(regionIndex: number, newText: string)` params.
- Get canvas from fabricRef, guard null
- Get store state (isProcessing, setIsProcessing, textRegions, setCanvasJson, setActiveTool, setTextRegions, setSelectedRegionIndex)
- If isProcessing, return early
- Get region = textRegions[regionIndex], guard undefined
- setIsProcessing(true)

Step 1 -- Create mask from bounding box (per D-07):
- Reset viewport, export canvas as base64 PNG (hiding any existing IText objects tagged with TEXT_REPLACE_TAG)
- const imageWidth/imageHeight from canvas dimensions
- const maskBase64 = createMaskFromBbox(region.vertices, imageWidth, imageHeight, 0.1)

Step 2 -- Inpaint to remove old text (reuse removeObject per D-07):
- const { cdnUrl } = await removeObject(base64Image, maskBase64)

Step 3 -- Load inpainted result onto canvas:
- const fabric = await import("fabric")
- const resultImg = await fabric.FabricImage.fromURL(cdnUrl)
- canvas.clear()
- canvas.setDimensions({ width: resultImg.width!, height: resultImg.height! })
- canvas.add(resultImg)

Step 4 -- Extract style and create IText (per D-08, D-10, D-11, D-12, D-13, D-14):
- const style = extractTextStyle(region.vertices) -- NOTE: skip imageData for now (client-side pixel sampling will be added if needed)
- For color extraction: try to use canvas getContext("2d").getImageData on the bounding box area from the ORIGINAL image (before inpainting). If not available, default to "#000000"
- const itext = new fabric.IText(newText, { left: region.boundingBox.x, top: region.boundingBox.y, fontSize: style.fontSize, fill: style.color, angle: style.angle, fontFamily: style.fontCategory === "serif" ? "Georgia, 'Times New Roman', serif" : style.fontCategory === "monospace" ? "'Courier New', monospace" : "Arial, Helvetica, sans-serif", editable: true, originX: "left", originY: "top", skewX: style.skewX, skewY: style.skewY })
- Tag the IText: (itext as any).TEXT_REPLACE_TAG = true
- Auto-fit: if itext rendered width > region.boundingBox.width * 1.15, scale fontSize down proportionally
- canvas.add(itext)
- canvas.setActiveObject(itext)
- canvas.renderAll()

Step 5 -- Update state:
- setTextRegions([]) -- clear detected regions (replacement done)
- setSelectedRegionIndex(null)
- toast.success("Text replaced -- adjust position/style, then click Apply to commit")
- In finally: setIsProcessing(false)
- Wrap in try/catch, toast.error on failure

**`handleTranslateAndReplace` callback** (per D-18, D-19, D-20, D-21):
Takes `(regionIndex: number, targetLang: string)`.
- Get region from textRegions[regionIndex]
- Call `const { translatedText } = await translateText(region.text, targetLang)`
- Call `handleReplaceText(regionIndex, translatedText)`

**`handleApplyText` callback** (per D-16):
- Flatten: export canvas to PNG data URL, create FabricImage from it, clear canvas, add flattened image
- Snapshot: setCanvasJson(JSON.stringify(canvas.toJSON()))
- Clear text state: setTextRegions([]), setSelectedRegionIndex(null)
- toast.success("Text committed to image")

**`handleCancelReplace` callback:**
- Remove any objects tagged with TEXT_REPLACE_TAG from canvas
- setTextRegions([]), setSelectedRegionIndex(null)
- setActiveTool("select")
- canvas.renderAll()

Export `TEXT_REPLACE_TAG = "__text_replace__"` constant.

Return: `{ handleDetectText, handleReplaceText, handleTranslateAndReplace, handleApplyText, handleCancelReplace }`

Create `src/components/editor/text-overlay-boxes.tsx` (per D-02, D-03, D-04):

"use client" directive.

This component renders HTML overlay boxes positioned absolutely over the canvas to highlight detected text regions. Uses CSS positioning (not Fabric.js objects) to avoid polluting the canvas/undo state.

Props: `{ fabricRef: RefObject<FabricCanvas | null> }`

Implementation:
1. Read `textRegions`, `selectedRegionIndex`, `setSelectedRegionIndex`, `activeTool` from useEditorStore
2. If activeTool !== "text-replace" or textRegions.length === 0, return null
3. Get canvas from fabricRef. If null, return null
4. Compute scale factor: the canvas element's CSS display dimensions vs the canvas internal dimensions. Use `canvas.getWidth()` for internal width and `canvas.lowerCanvasEl?.getBoundingClientRect()` for display dimensions. Scale = displayWidth / canvasWidth. Also get canvas element's offset (left, top) relative to its parent container.
5. Use a `useEffect` + `useState` pattern to track the canvas element's position and scale (re-calculate on resize or zoom changes). Listen for `useEditorStore.subscribe` for zoom changes.
6. Render a `<div>` with `position: absolute; pointer-events: none; top: 0; left: 0; width: 100%; height: 100%` that overlays the canvas area.
7. For each region in textRegions, render a box:
   - Position: `left: region.boundingBox.x * scale + offsetLeft`, `top: region.boundingBox.y * scale + offsetTop`
   - Size: `width: region.boundingBox.width * scale`, `height: region.boundingBox.height * scale`
   - Style: `border: 2px solid` with color based on selection state:
     - Selected (index === selectedRegionIndex): `border-blue-500 bg-blue-500/10`
     - Unselected with confidence >= 0.5: `border-green-500/70 bg-green-500/5`
     - Low confidence (<0.5): `border-yellow-500/50 bg-yellow-500/5 opacity-60`
   - `pointer-events: auto; cursor: pointer`
   - onClick: `setSelectedRegionIndex(index)`
   - Tooltip or small label showing truncated text content (first 20 chars) and confidence percentage
8. Use `position: relative` on the parent wrapper to enable absolute positioning of children

Keep this component lightweight -- it's just visual feedback, not canvas state.
