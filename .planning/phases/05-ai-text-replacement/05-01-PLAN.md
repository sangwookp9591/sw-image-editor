3. Create `src/lib/ai/ocr.ts` -- Google Cloud Vision REST API wrapper (per D-05, research recommends Cloud Vision):
   - Export `TextRegion` interface: `{ text: string; vertices: Array<{ x: number; y: number }>; confidence: number; locale?: string; boundingBox: { x: number; y: number; width: number; height: number } }`
   - Export `VisionAnnotation` interface matching the Vision API response shape (description, locale, boundingPoly.vertices)
   - Export `normalizeVertex(v: { x?: number; y?: number }): { x: number; y: number }` -- defaults missing x/y to 0 (Vision API omits zero values)
   - Export `parseTextAnnotations(annotations: VisionAnnotation[]): TextRegion[]` -- skip index 0 (full text), process rest into TextRegion array. For each annotation: normalize vertices, compute bounding box (minX, minY, width=maxX-minX, height=maxY-minY)
   - Export `callVisionOCR(base64Content: string): Promise<VisionAnnotation[]>` -- POST to `https://vision.googleapis.com/v1/images:annotate?key=${process.env.GOOGLE_CLOUD_VISION_API_KEY}` with body `{ requests: [{ image: { content: base64Content }, features: [{ type: "TEXT_DETECTION" }], imageContext: { languageHints: ["ko", "en", "ja", "zh"] } }] }`. Return `data.responses[0].textAnnotations ?? []`
   - If `GOOGLE_CLOUD_VISION_API_KEY` is not set, throw descriptive error

4. Create `src/lib/ai/text-style.ts` (per D-10, D-11, D-12, D-13):
   - Export `TextStyle` interface: `{ fontSize: number; color: string; angle: number; fontCategory: "serif" | "sans-serif" | "monospace"; width: number; height: number; skewX: number; skewY: number }`
   - Export `extractTextStyle(vertices: Array<{ x: number; y: number }>, imageData?: ImageData): TextStyle`:
     - fontSize: compute height from vertices[0] to vertices[3] distance, multiply by 0.75
     - angle: `Math.atan2(vertices[1].y - vertices[0].y, vertices[1].x - vertices[0].x) * (180 / Math.PI)`
     - color: call extractDominantColor if imageData provided, else default "#000000"
     - fontCategory: default "sans-serif"
     - width/height from bounding box
     - skewX/skewY: 0 default (best-effort per D-12)
   - Export `extractDominantColor(imageData: ImageData): string` -- histogram approach: iterate pixels in groups of 4 (RGBA), skip brightness >240 or <15, quantize to 32-step buckets, return hex of most frequent color
   - Export `createMaskFromBbox(vertices: Array<{ x: number; y: number }>, imageWidth: number, imageHeight: number, padding?: number): string` -- offscreen canvas, black background, white filled padded rectangle from bbox. Default padding 0.1 (10%). Return toDataURL("image/png")
