**Step 4: Create `src/lib/env.ts` — type-safe env validation (per D-08)**

Use `createEnv` from `@t3-oss/env-nextjs` with zod schemas:

- Map each to `process.env.*` in runtimeEnv

Use `skipValidation: !!process.env.SKIP_ENV_VALIDATION` so tests and CI can bypass.

**Step 5: Create `src/lib/db/schema.ts` — complete database schema (per D-09)**

Define these tables using `pgTable` from `drizzle-orm/pg-core`:

1. `users` — id (text PK), name (text not null), email (text not null unique), emailVerified (boolean default false), image (text nullable), createdAt (timestamp defaultNow), updatedAt (timestamp defaultNow)
2. `sessions` — id (text PK), expiresAt (timestamp not null), token (text not null unique), ipAddress (text nullable), userAgent (text nullable), userId (text FK->users.id cascade), createdAt, updatedAt
3. `accounts` — id (text PK), accountId (text not null), providerId (text not null), userId (text FK->users.id cascade), accessToken, refreshToken, idToken (all text nullable), accessTokenExpiresAt, refreshTokenExpiresAt (timestamp nullable), scope, password (text nullable), createdAt, updatedAt
4. `verifications` — id (text PK), identifier (text not null), value (text not null), expiresAt (timestamp not null), createdAt, updatedAt
5. `images` — id (text PK with crypto.randomUUID default), userId (text FK->users.id cascade), url (text not null), pathname (text not null), contentType (text not null), size (integer not null), createdAt (timestamp defaultNow)
6. `projects` — id (text PK with crypto.randomUUID default), userId (text FK->users.id cascade), name (text not null default "Untitled"), createdAt, updatedAt

Use exact column names from the research (snake_case in DB): `email_verified`, `ip_address`, `user_agent`, `account_id`, `provider_id`, `user_id`, `access_token`, `refresh_token`, `id_token`, `access_token_expires_at`, `refresh_token_expires_at`, `content_type`, `created_at`, `updated_at`, `expires_at`.

**Step 6: Create `src/lib/db/index.ts` — Drizzle instance**

```typescript
import { drizzle } from "drizzle-orm/neon-http";
import { neon } from "@neondatabase/serverless";
import * as schema from "./schema";

const sql = neon(process.env.DATABASE_URL!);
export const db = drizzle({ client: sql, schema });
```
