**1. Create `src/components/editor/hooks/use-object-eraser.ts`** — "use client" hook per D-11 through D-16:

Export constant: `MASK_TAG = "__mask__"` (must be exported for use-fabric.ts filter)

Accept `fabricRef: RefObject<FabricCanvas | null>`. Return `{ activateBrush, deactivateBrush, handleApply, handleCancel, brushSize, setBrushSize }`.

State: `brushSize` (default 30, range 5-100).

**`activateBrush()`** — per D-11, D-12:
1. Get canvas from fabricRef, bail if null
2. Dynamic import fabric: `const fabric = await import("fabric")`
3. Set `canvas.isDrawingMode = true`
4. Create `new fabric.PencilBrush(canvas)` and assign to `canvas.freeDrawingBrush`
5. Set brush color: `"rgba(255, 0, 0, 0.5)"` (red semi-transparent per D-12)
6. Set brush width to `brushSize`
7. Listen for `path:created` event on canvas: tag each new path with `path[MASK_TAG] = true` so it's filtered from undo

**`deactivateBrush()`:**
1. Set `canvas.isDrawingMode = false`
2. Remove `path:created` listener

**`handleApply()`** — per D-13, D-14, D-16:
1. Get canvas, bail if null or isProcessing
2. `setIsProcessing(true)`, `deactivateBrush()`
3. Collect mask paths: `canvas.getObjects().filter(obj => obj[MASK_TAG])`
4. If no mask paths, toast "Draw on the area to remove first", `setIsProcessing(false)`, return
5. Export original image as base64 (viewport-safe — same pattern as use-bg-removal):
   - Save viewport, reset to identity, toDataURL, restore
   - IMPORTANT: before exporting, temporarily hide mask paths (`obj.visible = false`) so they don't appear in the image export
   - Restore visibility after export
6. Export mask as black/white PNG using offscreen canvas (per Research pattern 6):
   - Create offscreen `document.createElement("canvas")` sized to canvas dimensions
   - Fill black background
   - For each mask path: set `obj.stroke = "white"`, `obj.fill = "white"`, render to offscreen context via `obj.render(ctx)` — OR simpler: iterate path objects, temporarily change their color to white, use `offscreen.toDataURL("image/png")`
   - Alternative simpler approach: create a temporary Fabric canvas, set background black, clone mask paths with white color, export toDataURL
7. Call `removeObject(base64Image, base64Mask)` server action
8. On success:
   - Remove all mask paths from canvas: `maskPaths.forEach(p => canvas.remove(p))`
   - Load result via `FabricImage.fromURL(cdnUrl)`, scale to canvas size
   - Clear canvas, add result, render
   - `setCanvasJson(JSON.stringify(canvas.toJSON()))` (undo snapshot for the full operation)
   - `setActiveTool("select")`
9. On error: toast error, keep mask paths so user can retry
10. Finally: `setIsProcessing(false)`

**`handleCancel()`:**
1. Remove all objects with `MASK_TAG` from canvas
2. `deactivateBrush()`
3. `canvas.renderAll()`
4. `setActiveTool("select")`

**Effect for brush size sync:** When `brushSize` changes and canvas.isDrawingMode is true, update `canvas.freeDrawingBrush.width`.
