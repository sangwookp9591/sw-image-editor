**1. Create `src/app/actions/ai-image.ts`** with "use server" directive containing three exported async functions:

- `removeBackground(base64Image: string): Promise<{ cdnUrl: string }>` — per D-01, D-02, D-05, D-15, D-17
  - Auth check: import `auth` from `@/lib/auth` and `headers` from `next/headers`, call `auth.api.getSession({ headers: await headers() })`, throw if null
  - Strip data URI prefix: `Buffer.from(base64Image.split(",")[1], "base64")`
  - Call `generateImage({ model: fal.image("fal-ai/bria/background/remove"), prompt: { text: "", images: [imageBuffer] }, providerOptions: { fal: { outputFormat: "png", syncMode: true } }, abortSignal: AbortSignal.timeout(55_000) })`
  - Upload result: `s3Client.send(new PutObjectCommand({ Bucket: BUCKET, Key: \`ai-results/${randomUUID()}.png\`, Body: image.uint8Array, ContentType: "image/png" }))`
  - Return `{ cdnUrl: getCdnUrl(key) }`

- `removeObject(base64Image: string, base64Mask: string): Promise<{ cdnUrl: string }>` — per D-03, D-05, D-16, D-17
  - Same auth check pattern
  - Convert both image and mask to Buffers
  - Call `generateImage({ model: fal.image("fal-ai/object-removal"), prompt: { text: "", images: [imageBuffer], mask: maskBuffer }, providerOptions: { fal: { syncMode: true } }, abortSignal: AbortSignal.timeout(55_000) })`
  - Upload to S3 and return CDN URL (same pattern)

- `generateBackground(prompt: string, aspectRatio?: string): Promise<{ cdnUrl: string }>` — per D-04, D-05, D-10, D-17
  - Same auth check
  - Call `generateImage({ model: fal.image("fal-ai/flux/dev"), prompt, aspectRatio: aspectRatio || "1:1", providerOptions: { fal: { outputFormat: "jpeg", syncMode: true } }, abortSignal: AbortSignal.timeout(55_000) })`
  - Upload result with `.jpg` extension, ContentType "image/jpeg"
  - Return CDN URL

Import `generateImage` from "ai", `fal` from "@/lib/ai/providers", `s3Client, BUCKET` from "@/lib/s3", `getCdnUrl` from "@/lib/cdn", `PutObjectCommand` from "@aws-sdk/client-s3", `randomUUID` from "crypto".
