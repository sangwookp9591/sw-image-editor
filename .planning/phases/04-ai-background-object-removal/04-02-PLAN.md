**1. Create `src/components/editor/hooks/use-bg-removal.ts`** — "use client" hook:

Accept `fabricRef: RefObject<FabricCanvas | null>` parameter. Return `{ handleRemoveBg, handleReplaceColor, handleReplaceGradient, handleGenerateBg, bgRemoved }`.

Constants at top of file:
- `BG_LAYER_TAG = "__bg_layer__"` — tag for background replacement objects (per Research pitfall 4)

**`handleRemoveBg()`** — per D-06, D-07, D-08, D-15:
1. Get canvas from fabricRef, bail if null or isProcessing
2. `setIsProcessing(true)`
3. Export canvas as base64 (viewport-safe per Research pattern):
   - Save `canvas.viewportTransform`, reset to `[1,0,0,1,0,0]`
   - Call `canvas.toDataURL({ format: "png", multiplier: 1 })`
   - Restore original viewport, `canvas.requestRenderAll()`
4. Call `removeBackground(base64)` server action
5. On success:
   - Load result: `const img = await FabricImage.fromURL(cdnUrl)` (dynamic import fabric)
   - Scale to canvas dimensions: `img.scaleToWidth(canvas.width!)` or `img.scaleToHeight(canvas.height!)` whichever fits
   - Clear canvas, add the result image, render
   - Trigger undo snapshot: `setCanvasJson(JSON.stringify(canvas.toJSON()))`
   - `setBgRemoved(true)`
6. On error: toast error via sonner
7. Finally: `setIsProcessing(false)`

**`handleReplaceColor(color: string)`** — per D-09, BG-02:
1. Get canvas, bail if null
2. Remove existing bg layer: find objects with `[BG_LAYER_TAG]` and remove them
3. Create `new fabric.Rect({ width: canvas.width, height: canvas.height, fill: color, selectable: false, evented: false })`
4. Tag it: `rect[BG_LAYER_TAG] = true`
5. `canvas.insertAt(0, rect)`, `canvas.renderAll()`
6. `setCanvasJson(JSON.stringify(canvas.toJSON()))`

**`handleReplaceGradient(colors: [string, string], direction: "horizontal" | "vertical")`** — per D-09, BG-02:
1. Same remove-existing pattern
2. Create Rect, set fill to `new fabric.Gradient({ type: "linear", gradientUnits: "percentage", coords: direction === "horizontal" ? { x1: 0, y1: 0, x2: 1, y2: 0 } : { x1: 0, y1: 0, x2: 0, y2: 1 }, colorStops: [{ offset: 0, color: colors[0] }, { offset: 1, color: colors[1] }] })`
3. Tag, insert at 0, render, snapshot undo

**`handleGenerateBg(prompt: string)`** — per D-04, D-10, BG-03:
1. `setIsProcessing(true)`
2. Calculate aspect ratio string from canvas dimensions (e.g., "16:9", "1:1", "4:3" — approximate to nearest standard)
3. Call `generateBackground(prompt, aspectRatio)`
4. Load result as FabricImage, scale to canvas size, tag with `BG_LAYER_TAG`, `selectable: false, evented: false`
5. Remove any existing bg layer, `canvas.insertAt(0, bgImage)`, render, undo snapshot
6. `setIsProcessing(false)`
