    3. Create `src/components/editor/hooks/use-editor-store.ts` (per D-14, D-15, D-16):
       - Zustand store wrapped with Zundo `temporal` middleware
       - Undoable state: `canvasJson: string | null` (serialized via canvas.toJSON())
       - UI state (NOT undoable, excluded via partialize): `activeTool: "select" | "crop" | "resize" | "pan"`, `zoom: number`, `isCropping: boolean`, `selectedPreset: string | null`, `imageUrl: string | null`, `imageName: string | null`
       - Actions: `setCanvasJson`, `setActiveTool`, `setZoom`, `setIsCropping`, `setSelectedPreset`, `setImageUrl`, `setImageName`
       - Zundo config: `limit: 30` (exceeds minimum 20 per D-16), `partialize` to only track `canvasJson`
       - Export the store as `useEditorStore`

    4. Create `src/components/editor/hooks/use-fabric.ts` (per D-02, D-03, D-04):
       - Custom hook `useFabric(canvasElementRef: RefObject<HTMLCanvasElement>, imageUrl: string)`
       - Inside useEffect, dynamically import fabric: `const fabric = await import("fabric")`
       - Create `new fabric.Canvas(canvasElementRef.current, { preserveObjectStacking: true })`
       - Store canvas instance in a ref (`fabricRef`)
       - Load image via `fabric.FabricImage.fromURL(imageUrl, { crossOrigin: "anonymous" })`
       - Set canvas dimensions to image native size, then fit viewport to container
       - Use ResizeObserver on container to handle window resize (debounce 150ms)
       - On `object:modified` event, serialize canvas to JSON and call `useEditorStore.getState().setCanvasJson(canvas.toJSON())`
       - Implement zoom: scroll wheel zooms centered on cursor via `canvas.zoomToPoint()`. Clamp zoom between 0.1 and 10.
       - Implement pan: space+drag via `canvas.relativePan()`. Set `canvas.defaultCursor = "grab"` when space held.
       - Dispose canvas on unmount
       - Return `{ fabricRef, isLoading }` where isLoading is true until image loads

    5. Create `src/components/editor/hooks/use-keyboard.ts` (per D-16):
       - `useKeyboardShortcuts()` hook
       - Ctrl/Cmd+Z: `useEditorStore.temporal.getState().undo()`
       - Ctrl/Cmd+Shift+Z: `useEditorStore.temporal.getState().redo()`
       - All shortcuts call `e.preventDefault()`

    6. Create `src/components/editor/hooks/use-clipboard.ts` (per EDIT-02):
       - `useClipboardPaste(fabricRef: RefObject<FabricCanvas | null>)` hook
       - Listen for `paste` event on window
       - Check `e.clipboardData.items` for image types
       - Convert to blob, create object URL, load via `FabricImage.fromURL()`
       - Add pasted image to canvas, update store with new canvas JSON
       - Clean up object URL after loading
